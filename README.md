# 💰 Income Expense Tracker

Bu proje, kullanıcıların **gelir ve giderlerini kolayca takip edebileceği**, finansal durumlarını görebileceği modern bir uygulamadır.  
Kullanıcı dostu arayüzü sayesinde işlemler hızlı ve anlaşılır bir şekilde gerçekleştirilebilir.

🔗 Canlı site: https://income-expense-tracker0.netlify.app

---

## 📌 Proje Özeti

- Gelir ve gider ekleme, silme, güncelleme ve listeleme
- Toplam bakiye hesaplama
- Finansal hareketleri anlık takip etme
- Kullanıcı dostu ve sade arayüz
- Responsive yapı (mobil uyumlu)

---

## 🚀 Kullanılan Teknolojiler

- React
- Next.js
- Tailwind CSS
- LocalStorage

---

## ⚙️ Kurulum

```bash
# Repo'yu klonla
git clone https://github.com/enesuludag0/income-expense-tracker.git

# Proje dizinine gir
cd income-expense-tracker

# Bağımlılıkları yükle
npm install

# Uygulamayı başlat
npm run dev
```
